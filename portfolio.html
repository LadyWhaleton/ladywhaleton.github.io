Work

Game Developer

Jan 2016 to Jul 2016


For 7 months, I worked as a game developer for ReCollect: The Game from UCR's Brain Game Center with a small team. ReCollect is a 2D game that revolves around memory based tasks such as n-back, item span, and multi-object tracking. We used Unity Game Engine to develop our game and programmed using C#. It is currently available on iTunes for all iOS devices.
I have added couple of features to the game, such as a popup message for leveling up and instructions for different game modes. I also implemented an additional feature called Sick Plants to the item span task. This feature makes the span task a little more challenging by requiring the player to select special plants ("Sick Plants") first.
